# Mental Wellbeing AI

An AI-powered web application that analyzes social media usage patterns and lifestyle factors to predict a person's mental wellbeing category.

The application uses Machine Learning to classify users into different wellbeing categories and provides a risk percentage along with a consultation recommendation.

##  Project Overview

In today's digital world, excessive social media usage can affect sleep, mood, anxiety, loneliness, and overall mental wellbeing.

This project uses machine learning to analyze different factors such as:

- Daily screen time
- Number of daily notifications
- Night-time screen usage
- Anxiety score
- Low mood score
- Average sleep duration
- Loneliness
- FOMO
- Social comparison
- Physical activity
- Digital detox habits
- Mental health support-seeking behavior

Based on these inputs, the system predicts the user's **wellbeing category** and provides an estimated risk percentage.

> This application is intended for educational and informational purposes only. It is not a medical diagnostic tool.
